# Computer networks | spring 2023 spbu

Целью курса является знакомство с базовыми принципами работы современных компьютерных сетей.

Курс построен вокруг пяти основных уровней Интернета: прикладной, транспортный, сетевой, канальный и физический. Будут рассмотрены базовые компоненты сети, важнейшие службы и протоколы, принципы взаимодействия сетей друг с другом.

Особое внимание на курсе уделено разработке сетевых приложений. По этой причине изложение материала идет сверху вниз: от верхнего уровня, доступного разработчикам, к нижним.
